WebSketch captures any web page into a compact semantic tree built from 23 UI primitives. Where raw HTML burns 50,000+ tokens of `div` soup, WebSketch produces 200--800 tokens of clean, labeled structure that any text model can reason about.

This handbook covers everything you need to work with WebSketch VS Code effectively:

- **[Getting Started](/websketch-vscode/handbook/getting-started/)** -- Install the extension, capture your first page, and configure settings.
- **[The Grammar](/websketch-vscode/handbook/grammar/)** -- The 23 roles, symbols, classifier tiers, and cleanup passes that produce the tree.
- **[Commands](/websketch-vscode/handbook/commands/)** -- All 6 commands, the 4 export views, and practical use cases.
- **[Ecosystem](/websketch-vscode/handbook/ecosystem/)** -- The WebSketch family of tools and how they share a common IR.

Whether you are pasting trees into ChatGPT, scaffolding components from a capture, or wiring up an AI agent, this handbook has the reference you need.
